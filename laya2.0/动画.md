# 图集动画运用
* 通过laya.display.Animation类的loadAtlas()方法加载角色的图集动画资源
* 图集动画使用loadAtlas()方法加载后，需要使用play()方法才可以播放。
* 如果图集中是独立的序列帧动画，直接使用play()方法就可以播放了。但是，将多个动画打包到一个图集里，如果要播放指定的动画，那就需要通过创建动画模板来实现。动画模板的方法是Animation.createFrames(),loadImages()
* animation animationBase
# 缓动动画
* 游戏开发中缓动动画比较常见，它是提升游戏UI体验的重要因素之一，例如对话框弹出、关闭，按钮的动效出现与消失，道具飞入背包等
* Tween缓动类的常用方法from()与to()
# 时间轴动画编辑器详解
* 在项目中使用时间轴动画主要分为三步。
>* 第一步：加载动画所需的图集。
>* 第二步：创建Animation实例，加载动画文件
>* 第三步：添加动画到舞台，播放动画
# 创建动效模板(EffectAnimation)
* 动效模板是基于时间轴的动画效果，通过预设动画效果，然后把效果附加给某个组件
